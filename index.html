Jeu du Memory

Vous retournez deux cartes de votre choix.

  • Si les cartes sont identiques, les cartes sont alors retirées et le vous pouvez rejouer.
  • Si les cartes ne sont pas identiques, les cartes se retournent face cachée de nouveau. Il vous faut alors rejouer.

Le jeu se termine une fois que toutes les paires ont été trouvées.
